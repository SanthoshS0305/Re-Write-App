### Example 1: Format a Title
1. Type your title
2. Select it
3. Choose "Georgia" from Font dropdown
4. Choose "24pt" from Size dropdown
5. Make it Bold (Ctrl+B)

### Example 2: Set Document Spacing
1. Select all text (Ctrl+A)
2. Click Spacing dropdown
3. Choose "1.5" for comfortable reading

### Example 3: Mix Fonts
1. Set body text to "Times New Roman" at 12pt
2. Set headings to "Arial" at 18pt
3. Each can have different formatting!

## ‚ùì Troubleshooting

**Q: Dropdown not working?**  
A: Make sure you have the latest changes. Run `npm install` in the frontend directory.

**Q: Font not displaying correctly?**  
A: Some fonts may have different names on different operating systems. We've included fallback fonts for compatibility.

**Q: Can I change font mid-sentence?**  
A: Yes! Select any portion of text and apply different formatting.

**Q: Are my formatting changes saved?**  
A: Yes! All formatting is preserved when you save revisions (both auto-save and manual).
