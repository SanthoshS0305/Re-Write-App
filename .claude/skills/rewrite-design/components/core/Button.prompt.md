A pill-shaped serif button for Re:Write actions; primary gets the green fill + hard black border.

```jsx
<Button variant="primary" size="lg">Login</Button>
<Button variant="secondary">Scene Manager</Button>
<Button variant="ghost">Cancel</Button>
```

Variants: `primary` (green action), `secondary` (raised teal), `ghost` (text), `danger` (red). Sizes: `sm`, `md`, `lg`.
