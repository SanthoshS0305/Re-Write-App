# Editor Features Guide

## Overview
The text editor now includes comprehensive formatting options including font selection, font size adjustment, and line spacing controls.

## Font Selection
The editor supports the following fonts:
- **Times New Roman** - Classic serif font
- **Cambria** - Modern serif font
- **Aptos** - Contemporary sans-serif
- **Merriweather** - Web-optimized serif (via Google Fonts)
- **Baskerville** - Traditional serif
- **Arial** - Standard sans-serif
- **Georgia** - Web-friendly serif
- **Comic Sans MS** - Casual font
- **Helvetica** - Clean sans-serif
- **Cochin** - Elegant serif
- **Garamond** - Classic book font
- **Futura** - Geometric sans-serif

### How to Use:
1. Select the text you want to format
2. Click the **Font Family** dropdown in the toolbar
3. Choose your desired font from the list
4. The font preview in the dropdown shows each font in its own style

## Font Size Options
Available font sizes range from 10pt to 36pt:
- 10pt, 11pt, 12pt (standard body text)
- 14pt, 16pt, 18pt (larger text)
- 20pt, 24pt, 28pt, 32pt, 36pt (headings and emphasis)

### How to Use:
1. Select the text you want to resize
2. Click the **Font Size** dropdown in the toolbar
3. Choose your desired size
4. The current size is displayed in the dropdown button

## Line Spacing
Four spacing options are available:
- **Single** (1.0) - Compact spacing
- **1.15** - Default spacing (Microsoft Word style)
- **1.5** - Reading-friendly spacing
- **Double** (2.0) - Academic/manuscript style

### How to Use:
1. Select the text or paragraph
2. Click the **Spacing** dropdown in the toolbar
3. Choose your desired spacing
4. The current spacing is shown in the dropdown button

## Additional Formatting Options
The toolbar also includes:
- **Bold** (Ctrl+B)
- **Italic** (Ctrl+I)
- **Headings** (H1, H2, H3)
- **Lists** (Bullet and Numbered)
- **Horizontal Rule**

## Technical Details
- Font settings are preserved when saving revisions
- All formatting is stored in the chapter content
- Fonts have fallback options for cross-platform compatibility
- Google Fonts integration for Merriweather ensures consistent appearance

## Tips
- Font and spacing changes can be applied to selected text or typed after selection
- To apply formatting to existing text, select it first before choosing from dropdowns
- Different fonts can be mixed in the same document
- Spacing affects the entire paragraph

## Browser Support
All fonts are supported across modern browsers (Chrome, Firefox, Safari, Edge)
